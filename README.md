# B-Book

B-Book is a breaking journal and arsenal manager for bboys and bgirls.

The goal is simple:
keep all your moves, combos, ideas, foundations, and progress tracking in one place.

Built with:
- React
- Vite
- Supabase

---

## Features

- Google Authentication
- Email Authentication
- Personal breaking dashboard
- Arsenal tracking
- Combo & set organization
- Foundations library
- Idea storage
- Session persistence
- Mobile-friendly UI

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/b-book.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_key
```

---

## Supabase Setup

1. Create a Supabase project
2. Enable Google provider
3. Copy API keys
4. Add `.env` values
5. Run the app
